# Custom Itinerary Travel

AI-powered travel itinerary platform with paywalled teaser pages and Wanderlog integration.

## Features

- AI-powered travel agents with RAG and MCP tools
- Paywalled itinerary teaser pages
- Wanderlog trip import and scraping
- Email booking system
- AI-enhanced activity descriptions
- Next.js 15 with TypeScript
- Supabase backend

## Setup

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Create `.env.local` file (see `.env.local` for required variables)

3. Start development:
   ```bash
   pnpm dev
   ```

4. Check for TypeScript errors:
   ```bash
   pnpm tsc --noEmit
   ```
